# 🎬 Trending Movies Web Application

A modern and responsive movie discovery application built with **React.js**, **Vite**, and **The Movie Database (TMDB) API**. 
Users can browse trending movies, manage a personalized watchlist, and organize movies by searching, filtering, and sorting within the watchlist.

## 🚀 Live Demo

**Live Website:** https://movies-app-chi-blond.vercel.app/

## ✨ Features

* Browse trending movies from TMDB
* View movie posters, ratings, and genres
* Add movies to a personalized watchlist
* Remove movies from the watchlist
* Search movies within the watchlist
* Filter movies by genre in the watchlist
* Sort the watchlist by rating and popularity
* Persistent watchlist using Local Storage
* Pagination for browsing trending movies
* Responsive design for desktop and mobile devices

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript (ES6+)
* Tailwind CSS
* Axios
* TMDB API
* Local Storage

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Madhansuriya31/Trending-Movies-Web-App.git
```

Navigate to the project folder:

```bash
cd Trending-Movies-Web-App
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add your TMDB API key:

```env
VITE_TMDB_API_KEY=YOUR_TMDB_API_KEY
```

Start the development server:

```bash
npm run dev
```

Open your browser:

```text
http://localhost:5173
```

## 🔗 API

This project uses **The Movie Database (TMDB) API** to fetch trending movie data.
